# static/

Statik dosyalar.

## pets/
Pet görselleri. Dosya adı formatı: `{egg_type}_{rarity}.png`

Örnek: `normal_common.png`, `golden_legendary.png`, `diamond_cosmic.png`

Desteklenen kombinasyonlar:
- egg_type: normal, golden, diamond
- rarity: common, rare, epic, legendary, mysterious, cosmic (sadece diamond)

Önerilen boyut: 256×256 px, PNG (transparan arka plan)
