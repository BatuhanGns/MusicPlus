# templates/

Flask Jinja2 şablonları.

## dashboard.html
Tek sayfa uygulama (SPA). Tüm görünümler bu dosyada:
- İstatistikler, İstatistikleriniz
- Hayvanlarım (pet sistemi)
- Profil (ayarlar, sync, XP/seviye)
- Playlist oluşturma
- Now playing + oynatma kontrolü
- AI sohbet

Tema değişkenleri CSS `:root` ile yönetilir.
Chart.js 4.4 dahil.
